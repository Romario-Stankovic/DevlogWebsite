About my site

This website is made using web technologies (HTML5, CSS3 and JavaScript). The topic of the website is Game Development, it has 4 main pages (Home, About, Devlogs and Contact) and 1 template page for posts. The "Home" page is the landing page of the website with introduction to what it is about and links to other pages. This page, the "About" page is used to give general information about me and this website, which you are reading right now :D. The "Devlogs" page is used for posts about progress in app or game development, and the "Contact" page is just that, a page which can be used to contact me.
In the header part of the page, there is a div which display the current weather in Belgrade using OpenWeatherAPI. The contact page has a form in which the user can enter his/her email, subject and message. The email field is validated through JavaScript and the message field requires at least one character to send. After validation, the page displays a popup indicating that the message is sent.
